👑 Royal Run (Unity)
📌 About The Project

Royal Run is a 3D endless runner game developed in Unity.
The player controls a character who must survive as long as possible by collecting coins and avoiding obstacles while the timer keeps decreasing.

🎮 Gameplay Features
⏳ Time-based survival system
🪙 Coin collection to increase score
⏱ Time boost pickups
🏃 Smooth player movement controls
💥 Game over when time runs out
🔊 Sound effects for coins and gameplay
🌀 Rotating pickups (coins, boosts)
🔁 Automatic restart after game over
💻 Scripts Included
🔹 PlayerController.cs

Handles player movement and controls.

🔹 GameManager.cs

Manages core game logic:

Timer system
Game over state
Restart system
UI updates
🔹 Pickups.cs

Base class for all pickups:

Rotation
Collision detection
Sound handling
🔹 Coin.cs
Increases score
Inherits from Pickups
